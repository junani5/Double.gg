# 🧥 WeatherFit: 개인 맞춤형 날씨 기반 옷차림 추천 서비스

## 🌟 프로젝트 소개

**WeatherFit**은 단순히 온도에 따른 옷차림을 제공하는 것을 넘어, **사용자의 개인적인 추위/더위 민감도를 학습**하여 가장 정확하고 만족도 높은 옷차림을 추천해주는 **개인 맞춤형 웹 애플리케이션**입니다.

Next.js 환경을 기반으로 개발되었으며, 사용자 피드백을 수집하여 머신러닝 모델을 지속적으로 개선하는 구조를 갖추고 있습니다.

### 주요 기능

* **위치 기반 날씨 조회:** 기상청 API를 통해 현재 위치의 실시간 기온, 습도, 풍속 등 상세 날씨 정보 제공.
* **규칙 기반 초기 추천:** 정의된 온도 구간별 옷차림 규칙(Rule-Based)에 따라 기본 추천 목록 제공.
* **🧠 개인 맞춤형 보정 (ML 적용):** 사용자의 과거 피드백(만족/불만족) 데이터를 머신러닝 모델이 학습하여, **추천 온도 보정 값**을 계산하여 맞춤형 결과 제공.
* **피드백 시스템:** 추천 결과에 대한 사용자 만족도 피드백을 수집하여 ML 모델 학습 데이터로 활용.
* **시간대별 예보:** 시간 흐름에 따른 기온 변화와 추천 옷차림 변화 시각화.

## 🏃 프로젝트 실행 방법

### 1. 환경 설정

1.  **레포지토리 클론:**
    ```bash
    git clone [프로젝트_레포지토리_URL]
    cd weather-fit
    ```

2.  **의존성 설치:**
    ```bash
    # Next.js/TypeScript 환경
    npm install 
    # 또는
    yarn install
    ```

3.  **API 키 및 환경 변수 설정:**
    프로젝트 루트 경로에 `.env.local` 파일을 생성하고, 다음 정보를 입력합니다.
    ```env
    # 기상청 공공 데이터 API 키
    KMA_API_KEY="당신의_발급_키"

    # ML 서버 주소 (로컬에서 실행 시)
    ML_SERVER_URL="[http://127.0.0.1:5000](http://127.0.0.1:5000)"
    ```

4.  **Python 환경 설정 (ML 서버):**
    별도의 Python 환경을 구축하고 필요한 라이브러리를 설치합니다.
    ```bash
    # Python 환경으로 이동 (예시)
    cd ml-server
    pip install -r requirements.txt
    ```

### 2. 프로젝트 구동

1.  **Python ML 서버 구동 (별도 터미널):**
    ```bash
    python app.py  # 또는 Flask/FastAPI 구동 명령어
    ```

2.  **Next.js 웹 서버 구동:**
    ```bash
    npm run dev
    # 또는
    yarn dev
    ```

웹 브라우저에서 `http://localhost:3000`으로 접속하여 서비스를 확인합니다.
